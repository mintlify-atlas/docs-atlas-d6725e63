## What you can build

Logo Generator produces three types of output in a single session:

- **SVG logo variants** — 6+ distinct designs based on your product's name, industry, and core concept, rendered as clean, scalable SVG files
- **Interactive HTML showcase** — a browser-ready page showing all variants side by side with hover effects and design rationale
- **High-end showcase images** — AI-generated 2K (16:9) presentation images that place your logo against one of 12 curated backgrounds, from cinematic dark styles to clean editorial light styles

## Get started

<Steps>
  <Step title="Install the skill">
    Run the automatic installer or clone the repository into your Claude Code skills directory.

    ```bash
    npx skills add https://github.com/op7418/logo-generator-skill.git
    ```
  </Step>
  <Step title="Install Python dependencies">
    Navigate to the skill directory and install the required packages.

    ```bash
    cd ~/.claude/skills/logo-generator
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Configure your Gemini API key">
    Copy the environment template and add your key for showcase image generation.

    ```bash
    cp .env.example .env
    # Edit .env and set GEMINI_API_KEY=your_key_here
    ```
  </Step>
  <Step title="Start a logo project">
    Restart Claude Code, then describe your product to begin.

    ```
    Generate a logo for my AI product called "DataFlow"
    ```
  </Step>
</Steps>

<Note>
  The Gemini API key is only required for Phase 4 showcase image generation. SVG logo generation and interactive HTML previews work without it.
</Note>
